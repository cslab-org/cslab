Assignment details
  1. Form groups of 3-4 members. In case the group size is 3, it is the instructors discretion for adding additional members.
  2. Selection of data: Select any data with the number of attributes not less than 20 and the number of sample points, not less than 10,000. The dataset should be multiclass (not binary) (Explore UCI data repository for a such of such data sets). In case the criterion for dataset is not met, then there should a valid and geniune reason for it. There should not be any collision in the selection of data.
  3. Fill the group details as well as the data details here.
    4. . Last date for entering details is 22nd September.
  4. First part deadline - Sept 30, 2016.
    A 1-2 page draft report consisting of information about the data, the various preprocessing steps engaged, along with some notable results. In case you have done some additional tasks, like classification, you can include them
  5. Final Submission of report - Oct 30, 2016. Evaluation of project - Nov 01, 2016
    The final report should include, all that what you done with the data. Nothing specific, but the evaluation would be based on how much you have played with the data, what interesting results, you have obtained from the data/classification, etc.
